# LibreHardwareMonitor Bundle

Download the latest portable release from:
https://github.com/LibreHardwareMonitor/LibreHardwareMonitor/releases

Extract these files into this directory:
- LibreHardwareMonitor.exe
- LibreHardwareMonitorLib.dll
- HidSharp.dll

These are not checked into git. The Tauri build bundles them into the app resources.
